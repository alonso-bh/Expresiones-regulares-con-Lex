
-----------------------------------------------------------------
-> INFORMACIÓN SOBRE LA EJECUCIÓN DEL PROGRAMA

El fichero "ejecutar", disponible en este repositorio, contiene las órdenes a ejecutar para poder probar el programa. Utilice el fichero de entrada "prueba_mc.txt", y como fichero de salida, donde se guardará la información extraída, el fichero llamado "salida.txt", que ahora contiene el resultado de una ejecución correcta realizada previamente. Este contenido se sobreescribirá automáticamente por el programa cuándo termine de ejecutarse.

-----------------------------------------------------------------
-> SOFTWARE NECESARIO 
   Los siguientes paquetes software básicos son necesarios para que se pueda probar
   la aplicación:
   1   make, g++
   2   flex++
   Estos paquetes se pueden instalar con el gestor de paquetes APT de Linux, o cualquier otro.

-----------------------------------------------------------------
-> RECOMENDACIONES
   Se recomienda utilizar distribuciones Linux para hacer las pruebas con la aplicación.
